# PC资源监视器  Computer Monitor
实时监视PC运行时的状态，如：

- CPU的占用率、温度
- GPU的占用率、温度
- 硬盘的读写速度
- 网络速度
- 内存的占用率

# 界面 UI

通过无框面板的方式置顶显示

可自定义面板内容
